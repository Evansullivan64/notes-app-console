# Notes app 

This project allows anybody to create and store notes

## Features

- Create notes
- Read notes
- Update any notes
- Delete any note
- extra validation for inputs
- list notes by title, starting letter, active and archived notes
### Persistence
- JSON
- XML
- YAML
